# ScoreLite

ScoreLite es una pequeña aplicación web para **consultar resultados y clasificación de fútbol**, centrada en LaLiga (Primera División española).  
La idea es tener una forma rápida y agradable de ver:

- **Partidos por jornada**: calendario de la liga y con marcadores.
- **Clasificación actual**: tabla completa con puntos, partidos jugados, diferencia de goles, etc.

La app está pensada como un **proyecto personal** para aprender y practicar React, así que la iré mejorando poco a poco:  
añadiendo más ligas, filtros, estadísticas y pequeños detalles visuales a medida que avance.
